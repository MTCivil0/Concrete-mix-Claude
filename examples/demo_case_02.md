# Demo Case 02 — High SCM Blend (F3 + C2, Fly Ash + Slag)

## Scenario
Parking structure in a northern climate with deicing chemicals.
Fly ash + slag blend for durability and cost reduction.

## Inputs
- f'c = 5000 psi
- Exposure: F3, C2
- Max aggregate: 3/4"
- Slump: 3-4"
- FM = 2.80, SG_CA = 2.65, SG_FA = 2.68
- Rodded density CA = 102 lbs/ft³
- Aggregate shape: Gravel with crushed faces
- Fly ash = 20%, Slag = 30%, PCC = 0%
- Air entrained: Yes

## Expected Results
- w/cm ≤ 0.40 (F3 + C2 limit)
- Air content: 6.0%
- Warning on total SCM % (50%) — verify early strength
- QC: ASTM C1202 chloride permeability recommended
